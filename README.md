# 小马逻辑

一款专为心爱之人制作的逻辑推理小游戏，在 6×6 的彩色网格中找出所有小马的位置。

## 游戏规则

1. **每种颜色只能放置 1 匹小马** - 每种颜色对应一个区域
2. **每行只能放置 1 匹小马**
3. **每列只能放置 1 匹小马**
4. **小马之间不能相邻**（包括斜对角方向）

## 操作说明

- **单击**：标记/取消该格子（画 ✕），用于排除错误位置
- **双击**：尝试在该位置放置小马
- **撤销**：撤销上一步标记操作
- **重开**：重新开始当前关卡
- **提示**：高亮显示一个小马的可能位置

## 游戏特性

- 🎯 **25 个关卡** - 逐步提升难度
- 💖 **好感度系统** - 每通关一次增加 5 点好感度
- ⭐ **星星收集** - 通关获得星星奖励
- ❤️ **生命机制** - 每关 2 条生命，错误放置会消耗生命
- 🔥 **连胜记录** - 追踪当前和最高连胜
- 🎉 **胜利动画** - 通关时播放彩带庆祝效果
- 💾 **自动存档** - 游戏进度自动保存到本地

## 运行方式

直接在浏览器中打开 `index.html` 文件即可开始游戏。

## 技术实现

- 纯原生 HTML/CSS/JavaScript 实现，无需任何依赖
- 响应式设计，完美适配移动端和桌面端
- 使用 CSS 动画提供流畅的交互体验
- N 皇后回溯算法生成有效的谜题布局
- 模拟玩家推理过程确保每个关卡都可解
- 本地存储（localStorage）持久化游戏进度
