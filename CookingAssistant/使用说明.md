# 烹饪助手使用说明

## 快速开始

### 第一步：安装 Ollama

1. 访问 Ollama 官网：https://ollama.com/
2. 下载并安装 Ollama（Windows 版本）
3. 安装完成后，Ollama 会在后台自动运行

### 第二步：启动服务

双击运行项目中的 `启动.bat` 文件

首次运行时会自动：
- 检查 Python 依赖并安装
- 下载 qwen2.5:7b 模型（约 4GB，首次需要几分钟）
- 启动 Web 服务

### 第三步：开始使用

1. 打开浏览器访问：http://127.0.0.1:5000
2. 即可开始使用烹饪助手

## 功能介绍

### 对话功能

在左侧对话框中输入问题，助手会基于知识库回答。

可以询问：
- 菜谱做法：如"红烧肉怎么做"
- 烹饪技巧：如"怎么给铁锅开锅"
- 食材选购：如"如何挑选新鲜的鱼"
- 营养知识：如"痛风病人不能吃什么"
- 厨具使用：如"不粘锅怎么保养"

### 筛选功能

右侧面板提供多维度筛选：

- 分类：川菜、粤菜、烘焙、家常菜等
- 难度：入门、中等、进阶
- 口味：咸鲜、麻辣、酸甜等
- 属性：平、凉、热、温

可以组合多个条件进行筛选。

### 主题切换

点击右上角设置按钮，可以切换界面主题颜色：
- 金色、翠绿、天蓝、紫罗兰、玫瑰

## 常见问题

### 启动失败怎么办

1. 确认已安装 Ollama
2. 运行 `ollama list` 检查模型是否下载成功
3. 手动下载模型：`ollama pull qwen2.5:7b`

### 回答很慢怎么办

本地模型运行速度取决于电脑配置，建议：
- 关闭其他占用内存的程序
- 使用 8GB 以上内存的电脑

### 想更换模型怎么办

编辑 `config.py` 文件：
```python
class Config:
    OLLAMA_MODEL = 'llama3.1:8b'  # 更换为其他模型
```

然后运行 `ollama pull 新模型名` 下载新模型。

2. Python 依赖是否完整安装
